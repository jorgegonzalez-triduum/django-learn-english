# OS requirements:
- Install the following dependencies:
    ``` bash

    sudo apt install \
        make \
        gcc \
        g++ \
        build-essential \
        gettext \
        python3-dev \
        python3-pil \
        zlib1g-dev \
        libpq-dev \
        libtiff5-dev \
        libjpeg8-dev \
        libfreetype6-dev \
        liblcms2-dev \
        libwebp-dev \
        graphviz-dev

    ```
# Python requirements:

- Install all dependencies using command `pip install -r requirements/base.txt`
